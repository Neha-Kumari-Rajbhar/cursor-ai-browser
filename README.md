# 🚀 AI Website Builder

## 📌 Overview

An AI-powered full-stack website builder that generates and previews React.js applications from natural language prompts using Gemini AI. The platform automates frontend code generation and provides a live sandbox environment for instant rendering and testing.


# ✨ Features

* 🤖 AI-based React.js code generation using Gemini AI
* ⚡ Real-time website preview with sandbox execution
* 🧠 Natural language to frontend conversion
* 🔄 Dynamic component rendering in-browser
* 📦 Full-stack MERN architecture
* 🛠 Modular and scalable project structure
* 🔒 Secure backend API handling


 # 🛠️ Tech Stack

## Frontend

* React.js
* TailwindCSS
* Vite

## Backend

* Node.js
* Express.js

## AI Integration

* Gemini AI API
* Prompt Engineering

## Database

* MongoDB


# 🏗️ System Architecture

User Prompt → Gemini AI → React Code Generation → Sandbox Execution → Live Preview

# 📌 Key Functionalities

### 🔹 AI Code Generation

Generates React.js frontend code from user prompts using Gemini AI.

### 🔹 Sandbox Execution

Runs generated React code securely inside the browser for live preview.

### 🔹 Real-Time Rendering

Instantly displays generated UI without manual setup or deployment.

### 🔹 Modular Architecture

Separated frontend, backend, and AI service layers for maintainability and scalability.


# ⚙️ Installation

```bash
# Clone repository
git clone <https://github.com/Neha-Kumari-Rajbhar/cursor-ai-browser/>

# Navigate to project
cd ai-website-builder

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install
```


# ▶️ Run Project

## Start Frontend

```bash
npm run dev
```

## Start Backend

```bash
npm start
```
